# 深蓝基础项目 1.0.0 发布了

Dark Blue（深蓝）是一个开源的脚手架项目。提供了一系列的基础功能，包括用户管理、权限管理、日志查看、字典管理、系统参数管理等功能，帮助开发者快速构建应用。

后端主要技术：

- JDK 21
- Spring Boot 3.x
- Spring Data JPA + QueryDSL

前端主要技术：

- Angular 18.x
- ng-zorro-antd
- echarts
