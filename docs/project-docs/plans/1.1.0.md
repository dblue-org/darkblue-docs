# 1.1.0

1.1.0 版本中主要是对历史版本中的问题进行修复，对前后端代码进行优化。不增加新功能。预计开始时间：2024-7-27

## 新特性

- 导航搜索功能实现
- 个人中心实现

## 优化

- 代码整体优化（前后端）：按照规范对代码进行修改（规范是在开发过程中完善的，因此会出现之前代码与规范不符的问题）。
- 优化部分功能重复调用后端接口的问题
